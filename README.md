# Python-Job-Listings-Scraper
In this project, you will build a Python web scraper that collects job listings from the Fake Python Jobs website. The scraper will extract information such as the job title, company name, location, and a link to the full job description.
Tools used in this project:
    python
    BeautifulSoup
    Pandas
    Requests
Main objectives of the project provided by roadmap.sh "https://roadmap.sh/projects/job-listings-scraper" is to learn the basics of web scraping and how to inspect HTML webpages, identify reusable patterns, and extract structured data using Python. You’ll practice working with third-party libraries, organizing scraped data, and exporting it for later analysis.
